Sobre o Projeto de Cadastro de Usuários

Este sistema foi desenvolvido com o objetivo de demonstrar práticas modernas de desenvolvimento web aplicadas à refatoração de código legado. Com uma abordagem modular e semântica, o projeto permite:

  • Cadastro e remoção de usuários de forma simples e dinâmica
  • Formulário de contato funcional e simulado
  • Estrutura responsiva e acessível
  • Separação clara entre HTML, CSS e JavaScript

Esta página "Sobre" é parte do esforço de criar um site completo, limpo e de fácil manutenção, respeitando os princípios de Clean Code, acessibilidade e usabilidade.
